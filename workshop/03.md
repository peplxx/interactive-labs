# Applications

One can be dynamically launch containers in the environment.

1. Example command: run an nginx server

    ```bash
    docker run --rm -it -p80:80 --name nginx --network lab_net nginx:alpine
    ```

1. Write `80` in the input field and click `+` to open it in a new tab.

1. Server is also reachable from other containers in the `lab_net` (e.g., the browser) by name

    ```bash
    ping http://nginx:80
    ```
